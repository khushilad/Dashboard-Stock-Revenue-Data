# Data Analysis Dashboard: Stock & Revenue Insights
This project focuses on analyzing historical stock prices and company revenue data to uncover trends, relationships, and insights.
## Key Features  

### Data Collection  
- Gathered data using **Web Scraping** techniques and **APIs** from multiple sources  

### Data Wrangling & Preparation  
- Cleaned, transformed, and structured raw datasets using Python libraries  

### Exploratory Data Analysis (EDA)  
- Identified trends, patterns, and anomalies in stock and revenue data  

### Statistical Analysis & Data Mining  
- Applied analytical techniques to derive meaningful insights  

### Data Visualization  
- Created visualizations (line charts, bar graphs, comparisons)  

### Interactive Dashboard  
- Built an interactive dashboard using IBM Cognos Analytics  

---

## Tech Stack  

- **Languages & Tools:** Python, SQL  
- **Libraries:** Pandas, NumPy  
- **Data Collection:** Web Scraping, APIs  
- **Visualization:** IBM Cognos Analytics  
- **Environment:** Google Colab
---
## What I Learned  
- End-to-end data analysis workflow  
- Combining multiple data sources (APIs + scraping)  
- Building interactive dashboards  
- Communicating insights through data visualization
